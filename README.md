リモートデスクトップセッションを自動で貼ってリモートからエクセルを起動するバッチファイル

実現したいこと：
リモートデスクトップ接続でエクセル起動用ユーザでログインしてエクセルを起動。起動が終わったらセッションを残したまま切断したい。
停止する時はエクセルを閉じるボタンで閉じてからログオフさせたい。

構成
コントロール側サーバ:psexecを設置(インストール不要)する必要あり
作業DIR:C:\TEMP
設置スクリプト:
excelstart.bat
excelstop.bat
IDGET.ps1
Win2012R2-excel.RDP

エクセルサーバ：ExcelViewerをインストールしておく必要あり

手順
開始
C:\TEMP\excelstart.bat  を実行

停止
C:\TEMP\excelstop.bat  を実行
